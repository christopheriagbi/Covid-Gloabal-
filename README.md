# 🌍 Global COVID-19 Analysis Dashboard | Power BI

# 📌 Project Overview

The **Global COVID-19 Analysis Dashboard** is an interactive Power BI project designed to analyze the worldwide impact of the COVID-19 pandemic. The dashboard provides insights into confirmed cases, deaths, infection rates, and vaccination progress across countries and continents through interactive visualizations.

It enables users to monitor pandemic trends, compare countries, and explore key health metrics to support data-driven decision-making.

---

# 🎯 Project Objectives

- Monitor global COVID-19 cases and deaths.
- Compare infection rates across countries.
- Analyze deaths by continent.
- Track the percentage of the population infected over time.
- Visualize the global spread of COVID-19 using maps.
- Support public health analysis through interactive dashboards.

|------|-------------|
| Total Cases | Total confirmed COVID-19 cases |
| Total Deaths | Total reported deaths |
| Death Percentage | Percentage of confirmed cases resulting in death |
| Countries | Number of countries included |
| Continents | Number of continents analyzed |
| Population Infected | Percentage of population infected |

# 💡 Business Questions Answered

- Which continent recorded the highest number of COVID-19 deaths?
- Which countries had the highest infection rates?
- How did COVID-19 spread over time?
- What percentage of each country's population was infected?
- Which countries experienced the highest mortality rates?

---

# 📈 Key Insights

- Europe recorded one of the highest COVID-19 death totals.
- Infection rates varied significantly across countries.
- Larger populations did not always correspond to higher infection rates.
- The global trend showed multiple waves of infections over time.
- Geographic analysis highlights regional differences in pandemic impact.
